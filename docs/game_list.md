# Native32 游戏清单

> Native32 是凌阳（Sunplus）为其 DVD 播放器和电视芯片开发的游戏格式，约 2008-2010 年广泛应用于便携式 DVD 播放器、车载后座显示器等设备。
>
> 游戏由"土豆工作室"专业团队制作，专门为 DVD 产品打造。
